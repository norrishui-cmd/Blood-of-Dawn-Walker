# Launch-Week Guide Templates

These templates are intentionally **not included in `sitemap.xml`**. They are editorial production assets for launch week. Copy a template into a real URL only after the topic is confirmed, tested, or clearly sourced.

## Rules

- Do not publish untested boss weaknesses, quest outcomes, build rankings, map routes, or achievement requirements.
- Keep spoiler-heavy details below spoiler warnings.
- Add a visible `Source And Verification` section to every published guide.
- Update `sitemap.xml`, hub links, and `updates/` when a template becomes a real page.
- Use official sources for release/store/platform claims and hands-on testing for gameplay claims.
