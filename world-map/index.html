FAQ

What should the map hub contain before release?
It should define regions, route-planning fields, location types, danger ratings, collectibles, shortcuts, and update rules.
Can location maps be indexed before testing?
Only if they provide honest status and useful planning. Exact coordinates and collectible routes require verification.
Which map pages matter most after launch?
Vale Sangora map, safe routes, danger zones, fast travel locations, hidden locations, resource locations, and boss locations.
