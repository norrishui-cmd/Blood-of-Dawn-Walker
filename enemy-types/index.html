The Blood of Dawnwalker Enemies and Monsters

Quick Answer

Confirmed enemy groups include vrakhiri, Blood Guards, kobolds, ancient undead, psoglavs, murohni, uriashi, likhos, and tatzelwurms. Only the first five currently have enough official detail for dedicated, indexable guides.

Confirmed lore and gameplay are separated from details that still require release-build testing.

Vampire-Aligned Enemies

The vrakhiri rule Vale Sangora under Brencis. Their servants include Blood Guards, humans altered through blood-bound rites, while psoglavs are canines corrupted by a vrakhiri fang. Murohni are described as failed vampires, but their origin, combat role, and appearance have not received enough official detail for a standalone page. These categories should not be collapsed into “vampire” because each represents a different relationship to blood corruption.

Mines, Ruins, and Wilds

Kobolds know the mines and attack distracted prey in groups. Ancient undead remain bound to duties inside forgotten ruins. Psoglavs roam the wilds with endless hunger. Habitat information gives players a useful pre-release expectation without pretending exact spawns are known. Post-launch, each entry should gain verified locations, weaknesses, variants, drops, and quest links only after hands-on evidence.

Folklore Creatures Still Awaiting Detail

Rebel Wolves has also named uriashi, likhos, and tatzelwurms. Uriashi are described elsewhere as mountain folk, while the others remain largely names in the official creature list. The site deliberately keeps them on this hub instead of publishing empty SEO pages. A dedicated URL should pass the concrete-answer gate only when official material or the released game supports several useful facts.

How This Bestiary Is Verified

Every indexable entry separates official lore from tactical inference. Familiar rules from tabletop games or other Slavic-folklore adaptations are not imported into Dawnwalker. Before release, no page should invent elemental weaknesses, loot tables, boss rewards, level ranges, or exact map coordinates. The linked entity pages cite Bandai Namco’s bulletins and the official gameplay recap directly.

How to Use This Pre-Release Guide

This entry records what Rebel Wolves and Bandai Namco have actually disclosed before launch. It does not convert folklore from another game into Dawnwalker mechanics, and it does not treat a trailer edit as proof of a fixed quest route. Exact map coordinates, dialogue outcomes, weaknesses, rewards, and missable conditions will be added only when the release build or a new primary source makes them verifiable. That distinction keeps the page useful now without forcing players to unlearn invented details later.
